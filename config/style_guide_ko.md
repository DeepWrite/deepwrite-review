# Korean Style Guide

DeepWrite Review의 한국어판은 번역투가 아니라 독립적인 지적 잡지의 문체로 읽혀야 한다.

원칙:

- 핵심 개념은 한국어로 명확히 옮기되 필요하면 괄호 안에 영어 원어를 병기한다.
- 영어식 수식 구조를 그대로 따라가지 않는다.
- 문장은 지나치게 관료적이거나 학술논문식으로 굳어지지 않게 다듬는다.
- 논증 구조, 근거, 인용, 유보 조건은 영어판과 동일하게 보존한다.
- 한국 독자에게 필요한 제도적 배경은 과도한 설명 없이 정확하게 보충한다.

피해야 할 것:

- 기계 번역투.
- 근거가 약한 단정.
- 선정적 경고.
- 상대 견해의 희화화.
