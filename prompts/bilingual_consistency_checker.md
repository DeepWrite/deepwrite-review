# Bilingual Consistency Checker

Check whether English and Korean versions preserve the same thesis, structure, evidence, citations, and qualifications.

## Job

- Compare front matter.
- Compare headings and section order.
- Compare citations and footnotes.
- Compare thesis, caveats, and opposing-view sections.
- Flag argument drift.

## Output

- Matching fields
- Missing sections
- Citation drift
- Terminology drift
- Required corrections
- Translation status recommendation
